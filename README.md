# ScaleBook

Twoje profesjonalne portfolio i techniczny dziennik budowy modeli.

## Kluczowe Funkcje:
- **Warsztat**: Dokumentacja postępów prac nad modelami (etapy budowy, zdjęcia, notatki).
- **Garderoba (Stash)**: Katalog modeli czekających na budowę z eksportem do PDF.
- **Notatnik Techniczny**: Miejsce na przepisy na farby i triki modelarskie.
- **Bezpieczeństwo**: Pełna synchronizacja z chmurą Supabase lub praca lokalna.
- **Kopia Zapasowa**: Eksport całej kolekcji (zdjęcia + dane) do jednego pliku ZIP.
- **Lokalizacja**: Pełne wsparcie dla języka polskiego i angielskiego.

## Technologie:
- Flutter / Dart
- BLoC (Cubit) dla zarządzania stanem
- Supabase (Auth, Database, Storage)
- GetIt (Service Locator)
- revenue_cat (Subskrypcje)
- in_app_review (System ocen)
